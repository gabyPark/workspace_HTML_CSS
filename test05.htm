1. input type="text" : (주관식) 한 줄 글상자

input 의 내용을 설명하는 텍스트는 태그로 묶어준다.
input 과 label 을 서로 연결시키기 위해 input에는 id를,
label에는 for를 추가하고 그 값을 서로 일치시킨다.

11. button ... /button : 전송하는 역할의 버튼

button ... /button 으로 입력한 데이터를 서버로 전송하는 역할의 버튼 생성.
button type="reset" 으로 페이지를 초기화 하는 역할의 버튼 생성.
button type="button"은 폼과 무관한 (스크립트로 기능추가) 일반 버튼 생성.
버튼 위의 글자는 시작과 종료 태그 사이에 적어준다.
* 시작과 종료태그 사이에는 다른 태그를 추가해서 아이콘을 넣거나 이미지를 넣거나
글자의 형태를 바꾸는 등의 추가 작업이 가능함.
