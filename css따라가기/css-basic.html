css적용법 3가지

  1. External CSS - 외부파일 방식. 스타일소스를 "*.css"라 따로 분리하여 만들고 html문서에 연결시켜 적용.
    통일성 있는 스타일 적용이 가능하며 유지보수에 유리.
  2. Internal CSS - 내장방식. 스타일소스를 html문서의 head안에 넣어서 적용.
    해당 웹페이지에만 스타일이 적용됨.
  3. Inline CSS - 인라인방식. 해당 태그 바로 뒤에 스타일소스를 적용.
    소스가 길어지고 통일성 있는 스타일 적용이 불가하며 유지보수에 불리.
    외부파일을 연결할 수 없거나 웹페이지의 본문만 수정 가능한 상황에서는 불가피하게 사용. (임대쇼핑몰이나 블로그 등)
    세가지 방식 중 CSS적용 우선 순위가 가장 높다는 장점이 있음.

css선택자 대표 3가지

  1. tag 선택자 - p { }
    해당 tag 전체에 적용됨. (자동, 강제 적용)
  2. ID 선택자 - #p { }
    구조적인 목적에 주로 사용. (ID값은 중복 적용이 불가하므로)
  3. Class 선택자 - .p { }
    디자인적인 목적에 주로 사용.
    (class값은 중복 적용이 가능하므로 여러 요소에 같은 스타일 적용 가능) (class값을 조합하여 사용 가능)
