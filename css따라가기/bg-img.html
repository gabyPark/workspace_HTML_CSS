background 배경

  1. background-color:색상명 또는 색상값; 배경 색상
  2. background-image:url("경로"); 배경 이미지

background-image의 세부 속성들

  • background-repeat: ; 배경 이미지의 반복 지정
    1. repeat 기본값. 양사방 반복
    2. repeat-x 가로로만 반복
    3. repeat-y 세로로만 반복
    4. no-repeat 반복 안함
  • background-size: ; 배경 이미지 크기 지정
    1. cover 이미지 하나로 영역을 다 채우기
    2. contain 이미지 하나를 온전히 보이기
    3. n n 가로 세로 순서로 직접 값 지정
  • background-position: ; 배경 이미지 위치 지정
    1. left/center/right top/center/bottom의 조합
    2. n n 가로 세로 순서로 직접 값 지정 (기본값은 좌측상단)
  • background-attachment: ; 배경 이미지의 스크롤 여부 지정
    1. scroll 내용과 함께 스크롤
    2. fixed 고정
    3. local

background-attachment 배경 이미지의 스크롤 여부

  1. scroll - 자신의 영역 스크롤바에는 고정, 브라우저 스크롤바에는 움직임
  2. fixed - 자신의 영역 스크롤바와 브라우저 스크롤바에 모두 고정
  3. local - 자신의 영역 스크롤바와 브라우저 스크롤바에 모두 움직임
