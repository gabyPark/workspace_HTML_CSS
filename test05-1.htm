2. html5에서 추가된 input type의 값들

모바일환경에서 사용자의 편리성을 추구하고
개발자 입장에서도 웹표준을 준수하여 개발하기 용이함.

input type="time"

시간을 (오전/오후 시분) 선택할 수 있는 폼요소

input type="date"

날짜를 (년월일) 선택할 수 있는 폼요소
min과 max 속성으로 최소값, 최대값을 지정할 수 있다.



input type="datetime-local"

날짜와 시간을(년월일 오전/오후 시분) 선택할 수 있는 폼요소

input type="week"

날짜를 (년주) 선택할 수 있는 폼요소

input type="month"

날짜를 (년월) 선택할 수 있는 폼요소

input type="tel"

연락처를 (전화번호) 입력할 수 있는 폼요소
모바일브라우저에서는 가상키패드가 숫자형식으로 바뀔 수 있고 전화 걸기가 가능하다.

input type="email"

연락처를 (이메일) 입력할 수 있는 폼요소
모바일브라우저에서는 가상키패드가 영문 입력이 쉽게 바뀔 수 있고
브라우저 자체에서 알맞은 이메일 형식인지 체크해줌 (@, @뒤의 내용)

input type="url"

연락처를 (웹주소) 입력할 수 있는 폼요소
모바일브라우저에서는 가상키패드가 영문 입력이 쉽게 바뀔 수 있고
브라우저 자체에서 알맞은 웹주소 형식인지 체크해줌 (프로토콜 https:// 나 http://의 유무)

input type="number"

숫자값을 입력(선택)할 수 있는 폼요소
min, max, step 속성으로 최소값, 최대값, 증감간격을 지정할 수 있다.





input type="range"

범위내의 값을 선택 할 수 있는 폼요소

input type="color"

색상을 선택 할 수 있는 폼요소

input type="search"

검색창을 지정 할 수 있는 폼요소 (삭제기능버튼)

3. 유용한 input의 속성들

1. autofocus

해당 요소에 자동으로 포커싱 되는 속성

2. autocomplete="on/off"

자동완성 기능을 켜거나 끄는 속성

3. required

필수입력으로 지정하는 속성

4. placeholder="..."

입력 예시나 설명 등을 표시하는 속성


5. readonly

읽기만 가능한 폼요소로 지정하는 속성 (수정불가, 값은 전송 가능)

6. disabled

비활성화된 폼요소로 지정하는 속성 (수정불가, 값 전송불가)
