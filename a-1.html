링크-1(상대경로) 링크-2

a 태그로 내부링크 걸기

 페이지 내에서 특정영역을 찾아서 이동하려면 특정영역에 주소로 활용할 수 있는 id를 추가한다.
                 (id값은 페이지 안에서 고유한 값이므로 주소로 활용 가능)
 링크를 걸 때  href속성의 값에(주소값) 해당 영역의 id를 표시하는데 이때, id값 앞에 기호#을 붙인다.

 <a href="#id값"> ... </a>
    ...
    ...
    ...
 <tag id="id값">
 
 


 다른 페이지(abc.html)의 특정영역을 찾아서 바로 이동하려면 해당 페이지의 원하는 영역에 주소로 활용할 수 있는 id를 추가한다.
 링크를 걸 때  href속성의 값에(주소값) 페이지주소#id을 붙인다.

  ( 링크 거는  페이지 )
 <a href="abc.html#id값"> ... </a>
    ...
