position 요소의 위치 지정방식

  • position:static; (기본값) 변화 없음
  • position:relative; 현재 위치에서 ~에 위치
  • position:absolute; 상위요소를 기준으로 ~에 위치
  • position:fixed; 브라우저를 기준으로 ~에 고정
  • position:sticky; relative와 fixed를 믹스. ~에서 고정
position:sticky;

거리값 (위치) 지정

  • left:n; 단위는 px,%,em, ... (음수값 가능)
  • right:n;
  • top:n;
  • bottom:n;

z-index:n; (음수값 가능)

겹치는 요소끼리의 위아래 순서 조정 가능. 값이 클수록 상대적으로 위로 올라옴. 소스상에 먼저 입력한 요소가 값이 작고 순서대로 1씩 증가함.

position:static; (기본값) 변화 없음

position:static; left: 800px;

position:relative; 현재 위치에서 ~에 위치

position:relative; left:800px;

position:relative; right: 300px;

position:relative; top:400px;

position:relative; bottom: 250px;

position:relative; left: 700px; bottom: 300px;

position:relative; right: 200px; bottom: 200px;

position:relative; left: 50%;

position:absolute; 상위요소를 기준으로 ~에 위치 (기준이 될 상위요소 position: relative; 설정)

position:absolute; left: 800px;

position:absolute; right:500px;

position:absolute; top:10px;

position:absolute; bottom:10px;

position:absolute; right: 0; bottom: 0;

position:absolute; top:-500px;

position:fixed; 브라우저를 기준으로 ~에 고정

브라우저창을 기준으로 해서 특정 위치에 고정이 됨. 스크롤과도 무관한 고정위치

position:sticky; relative와 fixed를 믹스. ~에서 고정

스크롤 되다가 특정 위치에서부터는 고정됨. position:sticky로 하려면 ! 자신을 둘러싼 다른 상위요소가 없어야 함.
