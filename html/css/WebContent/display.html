요소의 특징

  1. inline-한 줄에 나란히 오며 크기는 적용 불가, 상하 margin 적용 불가
  2. block-한 줄에 나란히 올 수 없고 크기와 모든 margin 적용 가능

display의 속성값들

  1. display:inline; block인 요소를 inline으로 바꾸기
  2. display:block; inline인 요소를 block으로 바꾸기
  3. display:inline-block; 요소를 inline과 block의 성격을 모두 가진 요소로 바꾸기
  4. display:none; 사라지게
